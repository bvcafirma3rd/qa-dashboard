Copy and share these links with each agent. They open directly to their personal QA view — no login needed.

ℹ️ How it works
Each link opens the dashboard filtered to that agent's data only. Agents see their own scores, category breakdowns, score trend, and review history — but cannot see other agents' data.

🔒 Sheet sharing required
For agents to load data, your Google Sheet must be set to "Anyone with the link can view". Go to your sheet → Share → Change to Anyone with the link → Viewer.

⚙️ Sheet Configuration

The sheet ID and GID are pre-filled for Path Social's QA sheet. Only change these if the sheet moves.
